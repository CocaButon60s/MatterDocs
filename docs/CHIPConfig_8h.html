This file defines default compile-time configuration constants for CHIP.

Package integrators that wish to override these values should either use preprocessor definitions or create a project- specific chipProjectConfig.h header and then assert HAVE_CHIPPROJECTCONFIG_H via the package configuration tool via –with-chip-project-includes=DIR where DIR is the directory that contains the header.
