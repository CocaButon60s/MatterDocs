static CHIP_ERROR chip::SetupPayload::FromStringRepresentation ( std::string  stringRepresentation,

static

Get a list of setup payloads from a string representation.

引数
[in]stringRepresentationThe string representing the payloads.
[out]outPayloadsOn success, the contents of this vector will be replaces with the list of parsed payloads. The result may have only one entry, or multiple entries if concatenated QR codes are used.

On failure, the value of the out param should not be relied on to be anything in particular.

◆ generateRandomSetupPin()

static CHIP_ERROR chip::SetupPayload::generateRandomSetupPin ( uint32_t &  setupPINCode)
static

Generate a Random Setup Pin Code (Passcode)

This function generates a random passcode within the defined limits (00000001 to 99999998) It also checks that the generated passcode is not equal to any invalid passcode values as defined in 5.1.7.1.

引数
[out]setupPINCodeThe generated random setup PIN code.
戻り値
Returns a CHIP_ERROR_INTERNAL if unable to generate a valid passcode within a reasonable number of attempts, CHIP_NO_ERROR otherwise
