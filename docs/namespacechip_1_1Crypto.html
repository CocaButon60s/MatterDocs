A function to add entropy sources to crypto library

This function can be called multiple times to add multiple entropy sources. However, once the entropy source is added, it cannot be removed. Please make sure that the entropy source is valid for the lifetime of the application. Also, make sure that the same entropy source is not added multiple times, e.g.: by calling this function in class constructor or initialization function.
