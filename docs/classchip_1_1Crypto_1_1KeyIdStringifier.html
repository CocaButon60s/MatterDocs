const char * chip::Crypto::KeyIdStringifier::KeyIdToHex ( ByteSpan  keyIdBuffer)
inline

Returns the null-terminated string buffer owned by the class containing converted KeyID.

LIFETIME NOTE: The last returned value from KeyIdToHex is valid until the next call.

This is optimized for standard 20-byte AKID/SKID but works for any length, truncating very long ones.

引数
keyIdBuffer- buffer of bytes of the key ID.
戻り値
pointer to class-owned storage of a null-terminated string in DCL format.
