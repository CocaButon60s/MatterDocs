◆ InitImpl()

CHIP_ERROR chip::Crypto::Spake2p_P256_SHA256_HKDF_HMAC::InitImpl ( )
overrideprotectedvirtual

Initialize underlying implementation curve, points, field elements, etc.

The implementation needs to:

  1. Initialize each of the points below and set the relevant pointers on the class: a. M b. N c. G d. X e. Y f. L g. Z h. V

    As an example: this.M = implementation_alloc_point();

  2. Initialize each of the field elements below and set the relevant pointers on the class: a. w0 b. w1 c. xy d. tempbn
  3. The hashing context should be initialized
戻り値
Returns a CHIP_ERROR on error, CHIP_NO_ERROR otherwise
