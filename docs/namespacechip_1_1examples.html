Initialize a LinuxCommissionableDataProvider from Linux common command-line options, falling back to PosixConfig, or hardcoded test parameters (if enabled). Handles passcode, verifier, salt, iteration count, and discriminator.

引数
provider- provider to initialize from command line arguments
options- LinuxDeviceOptions instance configured via command-line parsing
覚え書き
The following PosixConfig keys are used if present:
  • kConfigKey_SetupPinCode
  • kConfigKey_Spake2pVerifier
  • kConfigKey_Spake2pIterationCount
  • kConfigKey_Spake2pSalt
  • kConfigKey_SetupDiscriminator
