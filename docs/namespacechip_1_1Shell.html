typedef CHIP_ERROR chip::Shell::shell_command_iterator_t(shell_command_t *command, void *arg)

Execution callback for a shell command.

引数
commandThe shell command being iterated.
argA context variable passed to the iterator function.
戻り値
CHIP_NO_ERROR to continue iteration; anything else to break iteration.
