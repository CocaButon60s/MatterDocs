詳解

A minimal QR code setup payload generator that omits any optional data, for compatibility with devices that don't support std::string or STL.

◆ payloadBase38Representation()

CHIP_ERROR chip::QRCodeBasicSetupPayloadGenerator::payloadBase38Representation ( MutableCharSpan &  outBuffer)

This function is called to encode the binary data of a payload to a base38 null-terminated string.

The resulting size of the out_buf span will be the size of data written and not including the null terminator.

This function will fail if the payload has any optional data requiring TLV encoding.

引数
[out]outBufferThe buffer to copy the base38 to.
戻り値
CHIP_NO_ERRORif the method succeeded.
CHIP_ERROR_INVALID_ARGUMENTif the payload is invalid.
CHIP_ERROR_BUFFER_TOO_SMALLif outBuffer has insufficient size.
otherOther CHIP or platform-specific error codes indicating that an error occurred preventing the function from producing the requested string.
