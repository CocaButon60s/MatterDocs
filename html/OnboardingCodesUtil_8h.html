◆ EncodeQRCodeToUrl()

CHIP_ERROR EncodeQRCodeToUrl ( const char *  aQRCode,
size_t  aLen,
char *  aUrl,
size_t  aMaxSize 
)

Initialize DataModelHandler and start CHIP datamodel server, the server assumes the platform's networking has been setup already.

Method verifies if every character of the QR Code is valid for the url encoding and otherwise it encodes the invalid character using available ones.

引数
aQRCodeaddress of the array storing QR Code to encode.
aLenlength of the given QR Code.
aUrladdress of the location where encoded url should be stored.
aMaxSizemaximal size of the array where encoded url should be stored.
戻り値
CHIP_NO_ERROR on success and other values on error.
