template<size_t kCapacity>
class chip::Crypto::SensitiveDataFixedBuffer< kCapacity >

Helper class for holding fixed-sized sensitive data that should be erased from memory after use.

The sensitive data buffer is a fixed-length, fixed-capacity buffer class that securely erases the contents of a buffer when the buffer is destroyed.
