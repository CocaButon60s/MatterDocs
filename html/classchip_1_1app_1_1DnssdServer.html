CHIP_ERROR chip::app::DnssdServer::SetEphemeralDiscriminator ( Optional< uint16_t >  discriminator)

Overrides configuration so that commissionable advertisement will use an ephemeral discriminator such as one set for ECM. If the Optional has no value, the default basic discriminator is used as usual.

引数
[in]discriminatorEphemeral discriminator to override if it HasValue(), otherwise reverts to default.
戻り値
CHIP_NO_ERROR on success or CHIP_ERROR_INVALID_ARGUMENT on invalid value
