virtual CHIP_ERROR chip::Access::AccessRestrictionProvider::DoRequestFabricRestrictionReview ( const FabricIndex  fabricIndex,
uint64_t  token,
const std::vector< Entry > &  arl 
)
protectedpure virtual

Initiate a review of the access restrictions for a fabric. This method should be implemented by the platform and be non-blocking.

引数
[in]fabricIndexThe index of the fabric requesting a review.
[in]tokenThe unique token for the review, which can be matched to a review update event.
[in]arlAn optinal list of access restriction entries to review. If null, all entries will be reviewed.
戻り値
CHIP_NO_ERROR if the review was successfully requested, or an error code if the request failed.

chip::Access::ExampleAccessRestrictionProviderで実装されています。
