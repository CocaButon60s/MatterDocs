詳解

template<size_t kCapacity>
class chip::Crypto::SensitiveDataBuffer< kCapacity >

Helper class for holding sensitive data that should be erased from memory after use.

The sensitive data buffer is a variable-length, fixed-capacity buffer class that securely erases the contents of a buffer when the buffer is destroyed.
