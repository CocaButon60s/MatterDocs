A function that implements SHA-256 based HMAC per FIPS1981.

This implements the CHIP_Crypto_HMAC() cryptographic primitive in the the specification.

The out_length must be at least kSHA256_Hash_Length, and only kSHA256_Hash_Length bytes are written to out_buffer.

Error values are:

  • CHIP_ERROR_INVALID_ARGUMENT: for any bad arguments or nullptr input on any pointer.
  • CHIP_ERROR_INTERNAL: for any unexpected error arising in the underlying cryptographic layers.
