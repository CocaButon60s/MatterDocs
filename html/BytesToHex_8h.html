◆ HEX_ENCODED_LENGTH

#define HEX_ENCODED_LENGTH (   LEN)    ((LEN) << 1)

Computes the hex encoded length for a given input length. Left shift to generate optimized equivalent of LEN*2.

◆ HEX_MAX_DECODED_LENGTH

#define HEX_MAX_DECODED_LENGTH (   LEN)    ((LEN) >> 1)

Computes the maximum possible decoded length for a given hex string input length. Right shift to generate optimized equivalent of LEN/2.
