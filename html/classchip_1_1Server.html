詳解

The Server singleton class is an aggregate for all the resources needed to run a Node that is both Commissionable and mainly used as an end-node with server clusters. In other words, it aggregates the state needed for the type of Node used for most products that are not mainly controller/administrator role.

This sington class expects ServerInitParams initialization parameters but does not own the resources injected from ServerInitParams. Any object pointers/references passed in ServerInitParams must be pre-initialized externally, and shutdown/finalized after Server::Shutdown() is called.
