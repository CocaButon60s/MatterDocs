Create a ScopedMemoryString.

引数
[in]stringString to be copied.
[in]lengthLength to be copied. Like strncpy(), if the string is shorter than length, then the remaining space will be filled with null bytes. Like strndup() but unlike strncpy(), the result is always null-terminated.
